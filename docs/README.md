2. 了解 [版本信息](guides/VERSION.md)
3. 配置 [AI 功能](AI_README.md)

1. 阅读 [AI 功能说明](AI_README.md)

2. 阅读 [无障碍开发指南](guides/ACCESSIBILITY_GUIDE.md)（开发者）

2. 配置 [开发环境](guides/ENVIRONMENT_REQUIREMENTS.md)
3. 遵循 [无障碍开发指南](guides/ACCESSIBILITY_GUIDE.md)
4. 参考 [UI 设计变量](guides/UI_TOKENS.md)
